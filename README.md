
# RPG_CharacSheet

RPG_CharacSheet is a UI made to handle more easily the character sheets of a Role Playing Game with specific rules that a friend created. Current stable version is made with Qt for Python.

This program includes:

- an easy way to create a new character
<html>
    <img src="./.readme_images/characterCreator.png" width="770" height="460" alt="Character creator view" title="Character creator view"/>
</html>

- an intuitive user interface to visualize all the experience points spent and left to spend
<html>
    <img src="./.readme_images/caracTab.png" width="770" height="460" alt="Characteristics tab" title="Characteristics tab"/>
</html>

- a tab summing up all the necessary statistics for the game itself
<html>
    <img src="./.readme_images/resumeTab.png" width="770" height="460" alt="Summary tab" title="Summary tab"/>
</html>

- an Inventory tab to manage the character's inventory
<html>
    <img src="./.readme_images/inventoryTab.png" width="770" height="460" alt="Inventory Tab" title="Inventory Tab"/>
</html>

- an Abilities tab to manage the character's non-magical abilities
<html>
    <img src="./.readme_images/abilitiesTab.png" width="770" height="460" alt="Abilities Tab" title="Abilities Tab"/>
</html>

- a Spells tab to manage the character's magical abilities if this character is a mage
<html>
    <img src="./.readme_images/spellsTab.png" width="770" height="460" alt="Spells Tab" title="Spells Tab"/>

- a more general view to create abilities available to all characters
